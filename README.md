# GÜNDEM GLOBAL: AI-Powered Multilingual Platform 

GÜNDEM is a modern, responsive, and dynamic news platform built with Python. It seamlessly blends public relations and digital marketing dynamics with advanced software architecture to deliver news globally. 

This project demonstrates the integration of artificial intelligence tools and real-time translation services within a sleek, modern web interface.

##  Key Features
* **Dynamic Localization:** Instant UI and content switching between **Turkish, English, and Spanish**.
* **AI Integration:** Real-time news summarization powered by artificial intelligence.
* **On-the-fly Translation:** Automated translation of long-form articles into the user's preferred language using `deep-translator`.
* **Responsive UI:** A premium, "Silicon Valley" style magazine layout built entirely with **Flet** (Flutter for Python), ensuring a seamless experience across mobile and web platforms.
* **FastAPI Backend:** A robust API architecture to serve news data seamlessly to both web and mobile clients.
* **Bulletproof Architecture:** Defensive programming principles applied to prevent UI rendering crashes and ensure 100% stability.

##  Tech Stack
* **Language:** Python 3.x
* **Frontend / UI:** Flet (Python Framework based on Flutter)
* **Backend:** FastAPI, Uvicorn
* **Database:** SQLite3
* **AI & NLP Tools:** Deep-Translator, Threading (for non-blocking AI operations)

##  How to Run Locally

If you want to test this project on your own computer, follow these steps:

1. **Clone this repository:**
```bash
git clone https://github.com/alpsazlik/multilingual-portfolio-site.git
```

2. **Navigate to the project directory:**
```bash
cd multilingual-portfolio-site
```

3. **Install the required dependencies:**
```bash
pip install -r requirements.txt
```

4. **Run the API server:**
```bash
uvicorn api:app --reload
```

5. **Run the application (Open a new terminal window for this):**
```bash
python web_app.py
```
